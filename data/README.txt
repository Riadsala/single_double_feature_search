### buetti2019 ###

Contains original data from Buetti et al (2019) - available on OSF (https://osf.io/f3m24/).

### pilot ###

Contains data from our pilot experiment.
data_increased: all raw pilot data files.
initial_analysis.R: initial analysis of pilot data, creating accuracy_rt_data.txt.

### main ###

Contains all data from our main experiment.
proc_pavlovia_data.R: initial analysis of main data, creating accuracy_rt_data.txt.

spectrometer_readings.doc: contains relevant spectrometer readings for the experimental computer.
