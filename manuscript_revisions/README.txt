This folder contains all the files needed to recreate the second version of the Stage 2 manuscript.

Run stage_two_manuscript.tex to produce the document (generated html and a doc version are also available in the folder).

document_comparison.doc: comparing first version of the manuscript, and the second version.
Cortex Stage 2 response to reviewers.doc: response to reviewers document.
